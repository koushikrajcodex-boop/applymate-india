Bulk Scholarship Auto Import Bot

No browser scraping. No admin-page importing. The GitHub Action discovers scholarships and imports them to Firestore as review drafts.

This page is now only a dashboard for the automated Action. Run Bulk Scholarship Auto Import in GitHub Actions. It discovers scholarship sources, skips duplicates, imports draft records to Firestore, and writes the report shown here.

How it works

One-click bulk import flow

1. Run Action

GitHub Actions → Bulk Scholarship Auto Import → Run workflow.

2. Auto Import

The Action runs discovery and imports records into Firestore as draft/review.

3. Review Later

Imported records are not active by default. They stay safe as drafts until verified.

Required once: add FIREBASE_SERVICE_ACCOUNT_JSON and FIREBASE_PROJECT_ID in GitHub Actions secrets. Without that, the dashboard can show feeds, but Firestore import cannot happen.
