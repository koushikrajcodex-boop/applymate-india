# ApplyMate India Verification Workflow

This document explains the quality workflow for scholarship records before they are published to students.

## Goal

ApplyMate India should help students discover useful scholarships without showing unverified or incomplete records as active.

## Workflow

1. **Draft**
   - Add the scholarship record.
   - Keep it unpublished while details are incomplete.

2. **Review**
   - Check the official source link.
   - Confirm the deadline date.
   - Confirm eligibility details.
   - Confirm income rules.
   - Confirm education, category, gender, and disability rules.
   - Add a verification note.

3. **Publish**
   - Publish only when the record has the required quality fields.
   - Use the admin Verification Queue instead of the old Mark Active shortcut.

## Required fields before publishing

A scholarship should not be active unless these fields are complete:

- Name
- Official source link
- Deadline date when available
- Source name
- Eligibility note
- Income note
- Education list
- Category list
- Gender list
- Disability rule
- Application window status
- Verified-on date
- Verification note

## Application window values

Use one of these values:

- `open` — applications are currently open.
- `upcoming` — applications are expected but not open yet.
- `closed` — applications are closed.
- `verify` — details need to be checked again.

## Admin rules

- Do not publish a scholarship just because it looks useful.
- Do not publish without an official source link.
- Do not publish with unclear eligibility.
- Do not publish without checking the deadline.
- Keep doubtful records as draft or verify.

## Student-facing reminder

ApplyMate India is an informational student-help tool. Students should always check the official portal before applying.
