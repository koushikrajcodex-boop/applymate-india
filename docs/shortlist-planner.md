# Scholarship Shortlist Planner

The Shortlist Planner helps students decide which scholarship to apply for first.

## Scoring factors

The planner gives higher priority to scholarships that match:

- State
- Education level
- Category or student group
- Income range
- Document effort
- Official source clarity

## Important limitation

The planner is a decision-support tool. It does not confirm final eligibility. Students must verify official portals before applying.

## Current data source

The first version uses a small static scholarship list. Future versions can connect this to the verified Firestore scholarship database.
